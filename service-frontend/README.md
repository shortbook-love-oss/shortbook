## SvelteKit

### Creating a project

```bash
npm i
```

### Developing

```bash
npm run dev
```

Work at https://localhost:55022/

### Building

```bash
npm run build
node ./build
```

Work at https://localhost:55021/

## Prisma

### After changed Prisma schema

```bash
npm run migrate-dev --name="some-unique-name"
```
